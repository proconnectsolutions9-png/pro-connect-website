# Pro-Connect Solutions Website

A professional, responsive multi-page website for Pro-Connect Solutions, a small business AI agency.

## Pages

- **Home** (index.html) - Main landing page with hero section, services overview, testimonials, and CTA
- **Services** (services.html) - Detailed service descriptions with pricing tiers
- **About** (about.html) - Company mission, values, and team information
- **Contact** (contact.html) - Contact form with validation, contact information, and location

## Features

- **Mobile-first responsive design** - Fully responsive across all device sizes
- **Trust-focused corporate aesthetic** - Navy blue and silver brand colors
- **Form validation** - Client-side validation with real-time feedback
- **Sticky navigation** - Header stays visible while scrolling
- **Mobile menu** - Hamburger menu for mobile devices
- **SEO optimized** - Meta descriptions and semantic HTML structure

## Technology

- Pure HTML5 and CSS3
- No external dependencies or frameworks
- Vanilla JavaScript for form validation and mobile navigation
